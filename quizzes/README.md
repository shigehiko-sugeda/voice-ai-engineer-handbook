# 章別クイズ

各章を読んだ後に、用語、責務、ログ、車載ケースの理解を確認します。最初に回答を書き、必要な場合だけヒントを開き、最後に解答と解説を確認してください。

## 進め方

1. 対応章を読む。
2. ヒントを開かずに回答する。
3. 分からない場合はヒントを1つだけ確認する。
4. 全5問に回答してから解答と解説を開く。
5. 自己チェックで不足した章へ戻る。
