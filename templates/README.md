# 教材作成テンプレート

新しい章、クイズ、演習、設計レビュー資料を、既存教材と同じ構成で作るためのテンプレートです。`<...>`を対象内容へ置き換えて使用してください。

## 一覧

- [章テンプレート](chapter-template.md): `docs/`へ新しい章を追加する。
- [クイズテンプレート](quiz-template.md): `quizzes/`へ章別クイズを追加する。
- [演習テンプレート](exercise-template.md): `exercises/`へケース演習を追加する。
- [設計レビューテンプレート](design-review-template.md): 実務レビューの記録を作る。

## 共通ルール

- 教材本文に沿い、車載音声認識の文脈を優先する。
- 最初から答えを見せず、先にヒントや確認質問を置く。
- 入力、判断・処理、出力と、前後コンポーネントの境界を書く。
- 正常系だけでなく、Timeout、キャンセル、競合、Fallback、Recoveryを扱う。
- ログには関連ID、State、Event、時刻、失敗理由、版情報を含める。
- 設計レビューは、機能、品質、ログ、認証、安全性、運用に分ける。
- 認証の非公開仕様を推測せず、正式資料で確認する事項として扱う。

用語追加は専用テンプレートを使わず、[用語集](../docs/glossary.md)の該当カテゴリへ、既存形式に合わせて追記します。
